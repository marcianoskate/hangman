Hangman game
====

What is it?
----
this is the implementation of the Hangman game.

How to Run it
----
java -jar .\target\hangman.jar [filePath]

Documentation
----
Commands:
* While not playing:
** start: Starts a game
** help: Displays the command list
** exit: Ends the application

* While playing:
** exit: finishes the game and resets the state
